## 🌐 Deploy Rápido no Netlify

1. Aceda a [netlify.com](https://netlify.com) e crie uma conta gratuita
2. Clique em **"Add new site" → "Deploy manually"**
3. Arraste a pasta `hepomarket` completa para o Netlify
4. O site fica online em segundos com URL gratuito!

## 🐙 Deploy via GitHub

1. Crie conta em [github.com](https://github.com)
2. Crie repositório novo: `hepomarket`
3. Faça upload de todos os ficheiros
4. No Netlify, ligue ao repositório GitHub
5. Deploy automático a cada alteração!
